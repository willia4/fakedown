Fakedown

Markdown is a method of taking plain-text, human readable markup and turning it into HTML. But it has a nasty habit of adding html <p> tags whenever it sees a blank line.

This can be undesirable. Fakedown parses text using the standard Markdown rules except that it leaves your empty lines alone.
